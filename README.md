# like_zhihu
## 环境
* Python3
* MySQL

## 使用方法
使用manager.py对数据库进行初始化、迁移和升级
* python manager.py db init
* python manager.py db migrate
* python manager.py db upgrade

## 完成模块
* 注册
* 登录
* 发布问答
* 发布评论
* 问答列表
* 问答详情及评论
